# 初识

## Q：能干嘛？

* A：获取当前文件目录结构。

## Q：如何安装

* A：

  ````bash
  brew install tree
  ````

# 使用

## 参数说明

| 参数 | 描述                                                         |
| ---- | ------------------------------------------------------------ |
| -a   | 显示所有文件，包括隐藏文件（以  “.” 点开头的文件 ）          |
| -d   | 只显示目录                                                   |
| -f   | 只显示每个文件的全路径                                       |
| -i   | 不显示树枝，常与-f参数配合使用                               |
| -L   | level 遍历目录的最大层数，level 为大于0的正整数              |
| -F   | 在执行文件、目录、Socket符号链接、管道名称等不同类型文件的结尾，各自加上“\*”、 "/"、"="、"@"、" |

## Q：如何获取目录下指定层级目录的结构？

* A：格式：

  ````bash
  tree -L {层级}
  ````

  * 示例：

    ```bash
    tree -L 1
    ## 输出
    .
    └── src
    
    tree -L 2
    ## 输出
    .
    └── src
        └── components
    ```
